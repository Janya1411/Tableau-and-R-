Melbourne Livability Analysis Dashboard
=======================================

To launch the Melbourne Livability Analysis Dashboard, please follow the instructions below.

Prerequisites
Make sure you have R and RStudio installed on your system.

Installation
1. Install the required libraries by running the following commands (if required):

```R
install.packages("shiny")
install.packages("ggplot2")
install.packages("ggiraph")
install.packages("shinyjs")
install.packages("dplyr")
install.packages("DT")
install.packages("tidyr")
install.packages("sf")
install.packages("cicerone")
install.packages("reactable")
install.packages("reactablefmtr")

Launching the Dashboard
1) After installing the required libraries, open the RStudio/ any software that can run R file (VS code).
2) Navigate to the directory where you have saved the app.R file for the Melbourne Livability Analysis Dashboard.
3) Open the app.R file.
4) In the R script editor, click the "Run App" button. This will launch the Shiny application.
5) The default web browser should automatically open, displaying the Melbourne Livability Analysis Dashboard.

That's it! You should now have the Melbourne Livability Analysis Dashboard up and running for your analysis.
